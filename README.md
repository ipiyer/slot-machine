HTML5 slot-machine
============

HTML5 Slot machine game


## Install

```js
npm install
```

## Dev
``` js
grunt watch // recompiles sass and does jshint 
```
## test

```js
mocha-phantomjs test/index.html // open the file on the browser.
```
